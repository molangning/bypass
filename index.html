This website takes advatages of some device management profiles that does not take into account of certain redirect code, such as code 302.
The website might be able to "redirect" you to websites previously not allowed by the management, but sometimes they might not work.
The free tool works like link shorteners without the shortening part, as we cannot store any information permanently.To create a permalink, use this format: https://bypass--redirect-project.repl.co/redirect.php?url=www.google.com and replace google.com with the link you want to go to.
